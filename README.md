# 💼 Cartão de Visita Digital — CLI

Gerador de cartão de visita profissional em terminal, com borda e centralização dinâmicas baseadas nos dados informados pelo usuário.

Projeto autoral desenvolvido de forma independente — a ideia foi sugerida por um mentor como primeiro desafio prático, mas toda a lógica e construção do código foram desenvolvidas de forma completamente autoral, incluindo a solução de borda dinâmica elaborada antes mesmo de o recurso ser apresentado formalmente.

---

## 🚀 Tecnologias & Conceitos

- **`len()`** para medição dinâmica do conteúdo
- **Multiplicação de strings** `"=" * n` para borda adaptável
- **`.center()`** com referência dinâmica para centralização automática
- **f-strings** para interpolação de variáveis
- **Variável de referência** reutilizada para manter consistência visual

---

## 🗂️ Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| **Coleta** | Nome, profissão, empresa, e-mail e telefone |
| **Borda dinâmica** | Tamanho se ajusta automaticamente ao conteúdo |
| **Centralização** | Todos os campos centralizados com base na borda |

---

## ▶️ Como executar

```bash
git clone https://github.com/seu-usuario/cartao-de-visita-cli
cd cartao-de-visita-cli
python cartao_de_visita.py
```

# 💼 Digital Business Card — CLI

A terminal-based professional business card generator with dynamic border and centering based on user-provided data.

An independent project — the concept was suggested by a mentor as a first practical challenge, but all logic and code were developed entirely and autonomously by the author, including a dynamic border solution built before the feature was formally introduced.

---

## 🚀 Technologies & Core Concepts

- **`len()`** for dynamic content measurement
- **String multiplication** `"=" * n` for adaptive border
- **`.center()`** with dynamic reference for automatic centering
- **f-strings** for variable interpolation
- **Reference variable** reused for visual consistency

---

## 🗂️ Features

| Feature | Description |
|---|---|
| **Data collection** | Name, profession, company, email and phone |
| **Dynamic border** | Size auto-adjusts to content length |
| **Centering** | All fields centered relative to the border |

---

## ▶️ How to run

```bash
git clone https://github.com/seu-usuario/cartao-de-visita-cli
cd cartao-de-visita-cli
python cartao_de_visita.py
```
